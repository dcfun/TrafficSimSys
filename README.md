# TrafficSimSys
两路口 交通仿真系统

系统目标：  
#模拟未来交通 无人驾驶  

路上没有红绿灯了，人也不必考驾照了，同时 路上再也不会堵车了。。。  

##交通规则：  
只能直行 或 右转，走最短路径、、到达目的地。  

*判断相撞函数1*：
